# Keymap Toolkit

A Blender add-on that provides custom keymap configuration.

## Overview

Keymap Toolkit is an add-on for customizing Blender's keymap settings. It provides preset keymap configurations to enable efficient workflows.

## Features

- Custom keymap configuration
- Keymaps for multiple space types
- Intuitive key binding settings

## Installation

1. Launch Blender
2. Open `Edit > Preferences > Add-ons`
3. Click `Install...`
4. Select the ZIP file for this add-on
5. Search for "Keymap Toolkit" in the add-on list and enable the checkbox

## Usage

Once the add-on is enabled, the keymap configuration will be automatically applied. You can view the keymap details in `Edit > Preferences > Keymap`.

## Requirements

- Blender 4.2.0 or later
