# GZR Keymap

Custom key maps for view navigation in Blender.

## Features

This add-on provides custom key mappings for viewport navigation:

### 3D View
- **Rotate View**: Alt + Left Mouse Button
- **Move View**: Alt + Middle Mouse Button
- **Zoom View**: Alt + Right Mouse Button

### 2D View
- **Pan View**: Alt + Left/Middle Mouse Button
- **Zoom View**: Alt + Right Mouse Button

### Image Editor
- **Pan View**: Alt + Left/Middle Mouse Button
- **Zoom View**: Alt + Right Mouse Button

## Installation

1. Download the add-on from the releases page
2. In Blender, go to `Edit > Preferences > Add-ons`
3. Click `Install...` and select the downloaded zip file
4. Enable the add-on by checking the checkbox next to "GZR Keymap"

## Usage

After installation, the custom key mappings will be active. You can view and customize the keymaps in the add-on preferences.
